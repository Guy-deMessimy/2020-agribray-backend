- passport-jwt: 4.0.0
- passport-local: 1.0.0

# SETUP :

To run this project, install it locally using npm:

```
    $ git clone https://github.com/Guy-deMessimy/2020-agribray-backend.git
    $ cd 2020-agribray-backend
    $ npm install
    $ npm start
```
